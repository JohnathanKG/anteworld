How to use OT knob bones

1) Select armature and make sure you are using Edit mode
2) Select bone in armature
3) Open "Bone" context in "Properties" panel
4) See section "Outerra Knob Element" in list

Note 1: You can direct edit bone's custom properties, this plugin is just more sophisticated editting tool
Node 2: For more info about properties that can be set, visit https://github.com/Outerra/anteworld/wiki/Vehicle-IK:-Interactive-elements
